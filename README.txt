# 🍛 Food Delivery Website

A modern and responsive food delivery web application inspired by platforms like Zomato and Swiggy. Users can explore menus, add items to a cart, and simulate placing orders.

---

## 🚀 Live Demo

👉 https://github.com/Sravankumar12367/food-delivery-website.git

---

## 📸 Features

* 🔐 User Registration & Login (LocalStorage)
* 🍽️ Interactive Food Menu
* 🛒 Add to Cart with Quantity Controls
* 📊 Real-time Cart Sidebar with Total Price
* ✅ Order Placement Simulation with Order ID
* 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **LocalStorage**
